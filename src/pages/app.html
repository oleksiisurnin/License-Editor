License List
Add New
Here you can edit licenses. Type a license name and leave a link to the license document. If you don’t know what “slug” is, type the word “License”. Don’t forget to assign at least one asset attribution to it. You can also type here any text you want and leave any link, but make sure that it’s not against the Facebook Policy and Rules.
Asset Attributions
Add New
Here you can assign asset attributions to licenses. Each license should have at least one asset attribution under it. Create new asset attribution for each asset, specify the asset's name, author and URL that will open when the user clicks at this license item. Select the corresponding asset and assign a license using dropdown lists.
